# Auto Install Rule

作業中に必要なCLIツールやフレームワークが未インストールの場合、
**ユーザーの許可なく自動でインストールして作業を継続する**。

## 対象ツール例

- Vercel CLI (`npm install -g vercel`)
- その他のnpmグローバルパッケージ
- 開発に必要なフレームワーク・ライブラリ

## ルール

// turbo-all

1. ツールが未インストールであることを検知
2. 適切なパッケージマネージャーでインストール
3. 作業を継続

## 注意

- インストールは `npm install -g` または `npm install` で行う
- インストール結果（成功/失敗）は作業サマリーに記載する
